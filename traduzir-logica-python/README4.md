# 🧠 EX4 - Traduzindo Lógica para Python

Projeto acadêmico desenvolvido com o objetivo de praticar a tradução de algoritmos e pseudocódigos para Python, utilizando estruturas condicionais, loops, funções e entrada de dados.

---

# 📖 Descrição

O projeto reúne diferentes exercícios de lógica de programação implementados em Python, abordando situações práticas como:

- Processamento de vendas
- Análise climática
- Sistema de notas escolares
- Simulação de investimentos

Cada exercício trabalha conceitos fundamentais da programação estruturada.

---

# ⚙️ Funcionalidades

## 🛒 Processamento de Vendas
- Cadastro de produtos
- Cálculo de subtotal
- Aplicação de descontos automáticos
- Resumo da compra

## 🌡️ Análise Climática
- Cálculo da média semanal de temperatura
- Identificação de dias quentes
- Alertas de temperatura extrema

## 🎓 Sistema de Notas
- Cadastro de alunos
- Cálculo de médias
- Classificação entre aprovado, recuperação e reprovado

## 💰 Simulador de Poupança
- Simulação de juros mensais
- Controle de aportes
- Meta financeira automática
