# Sistema de Triagem Médica Automatizada

## 📖 Descrição
Projeto desenvolvido na disciplina de Engenharia de Soluções Lógicas com o objetivo de criar um sistema de triagem médica automatizada utilizando algoritmos, fluxogramas e lógica de programação.

O sistema analisa sinais vitais do paciente, como:
- Pressão arterial
- Frequência cardíaca
- Temperatura corporal

Com base nesses dados, o algoritmo classifica automaticamente o atendimento entre **clínico** e **emergência**.

---

## ⚙️ Funcionalidades
- Leitura de sinais vitais do paciente
- Classificação automática do atendimento
- Estrutura lógica em fluxograma
- Implementação em Portugol
- Testes de mesa para validação

---

## 🛠️ Tecnologias Utilizadas
- Portugol
- Fluxogramas
- Lógica de Programação

---

## 🎯 Objetivo
Automatizar o processo inicial de triagem médica, auxiliando na tomada de decisão de forma rápida, simples e organizada.
