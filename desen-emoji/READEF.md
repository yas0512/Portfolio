#  Gerador de Emojis

Projeto desenvolvido em Python com o objetivo de gerar e manipular emojis de forma simples e interativa, utilizando lógica de programação e automação.

---

#  Descrição

O sistema permite trabalhar com emojis através de códigos e estruturas em Python, explorando conceitos de manipulação de texto, entrada de dados e organização lógica.

O projeto foi desenvolvido como prática de programação e experimentação com caracteres Unicode e bibliotecas Python.

---

#  Funcionalidades

- Geração de emojis
- Manipulação de caracteres Unicode
- Exibição dinâmica de símbolos
- Interação com o usuário
- Estruturas condicionais e repetição

---

#  Tecnologias Utilizadas

- Python 3
- Jupyter Notebook

---

#  Conceitos Aplicados

- Strings
- Unicode
- Estruturas condicionais
- Loops
- Entrada e saída de dados
- Manipulação de caracteres especiais

---

#  Objetivo

Praticar lógica de programação e manipulação de texto em Python através da criação de um sistema simples de geração de emojis.
